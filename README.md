# Travel Wishlist - Web Development Final Project

##  About
 The website lets you add, track, and manage travel destinations.

##  What It Does
- Add new travel destinations
- Mark places as "Want to Visit" or "Visited"
- Search and filter your list
- View statistics about your travel goals
- Data saves automatically in your browser
